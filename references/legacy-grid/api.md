## API

### Row Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| gutter | `number \| string \| [number, number] \| [string, string]` | `0` | `水平间隔` 或者 `[水平间隔, 垂直间隔]` |

### Col Props

| 名称   | 类型     | 默认值 | 说明 |
| ------ | -------- | ------ | ---- |
| span   | `number` | `1`    |      |
| offset | `number` | `0`    |      |
| push   | `number` | `0`    |      |
| pull   | `number` | `0`    |      |
